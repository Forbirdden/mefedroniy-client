# Цветные юзернеймы

Имена пользователей отображаются разными цветами в зависимости от клиента, через который пользователь подключён.

## Mefedroniy
- Регулярка: `\u00B0\u0298<(.*?)> (.*)`
- Цвет: Светло-пурпурный
- Пример: `°ʘ<ник> текст`

## bRAC

- Регулярка: `\uB9AC\u3E70<(.*?)> (.*)`
- Цвет: Зелёный
- Пример: `리㹰<ник> текст`

## CRAB

- Регулярка: `\u2550\u2550\u2550<(.*?)> (.*)`
- Цвет: Светло-красный
- Пример: `═══<ник> текст`

Tower

- Регулярка: `\u25B2<(.*?)> (.*)`
- Цвет: Белый
- Пример: `▲<ник> текст`

clRAC

- Регулярка: `<(.*?)> (.*)`
- Цвет: Голубой
- Пример: `<ник> текст`
